    Purpose

    The purpose of this project was to create and develop a digital game where collaboration is encouraged between the players. The game is created for teams of four players, where the key to success is close collaboration.

    In order to implement collaboration in a game, it was efficient to carefully study the topic of collaboration and the different game patterns that encourage it. The result is a fully functional computer game with elements of collaboration such as interdependence and asymmetry. It was also confirmed that the purpose of the project was achieved through user tests performed on students aged 10 to 11.

    Result

    The final game is an endless runner type of game (but where the playtime is quite short) where one player is controlling the speed and direction by running on step pads, and the other player is dodging obstacles by pressing the right buttons at the correct time.

    As this game is a collaborative game it is important to find a good balance between the players' roles. One way this has been taken into consideration in the game is by giving the players asymmetric abilities. Asymmetric abilities are a way of giving every player a feeling that they are important and contribute to the result in their own unique way if it is implemented in a good way. In this game, the asymmetric abilities are quite differentiated from each other and the players need to cooperate and use their abilities to be able to reach success.

    The game was placed at an exhibition where people got to test it during two days. During the exhibition we got a lot of positive response and a verification that we managed to create a fun collaborative game where the players had to move but without it feeling forced.
